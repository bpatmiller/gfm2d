Detailed Description

The main simulation class that defines our computational domain.

It is described spatially by a cell size (h) and a number of cells in both dimensions (sx, sy).

stores velocity and pressure, a list of fluids contained in it, a solid phi describing boundary locations a map from voxels to fluid type

void Simulation::run ( )

Runs the main simulation loop.

Exports simulation data at each timestep, and breaks the timestep into substeps based on the CFL condition time_elapsed - current runtime of the simulation max_t - total amount of time the simulation will run timestep - amount of time between "frames" t - tracks the amount of time traversed in a given frame substep - a length of time given by cfl()
