Detailed Description

template<class T>
class Array2< T >

A 2d array template with a consistent spatial indexing scheme for use in a collocated grid.

For translation between worldspace and gridspace, we use a linear offset. The leastmost worldspace coordinate considered in our universe is (0,0)
